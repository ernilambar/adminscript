WordPress plugin to add JS in the admin panel.

== Description ==

Adminscript is a simple WordPress plugin to add Javascript in the admin panel. Custom JS is hooked to admin footer.

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' Plugin Dashboard
1. Select 'adminscript.zip' from your computer
1. Install
1. Activate the plugin on the WordPress Plugin Dashboard

= Using FTP =

1. Extract 'adminscript.zip' to your computer
1. Upload the 'adminscript' directory to your '/wp-content/plugins/' directory
1. Activate the plugin on the WordPress Plugins dashboard
